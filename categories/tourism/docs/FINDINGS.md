## 二、核心发现一：AI 心里的"上海"长什么样（景点占答排行）

**外地游客（A，78 条）—— 经典地标霸榜：**

| 景点 | 占答率 | 首选率 | 地位 |
| --- | --- | --- | --- |
| **外滩** | **51.3%** | **39.7%** | 🥇 绝对霸主，AI 的"上海第一答案"，几乎问什么都先说它 |
| 豫园 | 46.2% | 7.7% | 常被提但很少排第一 |
| 东方明珠 | 39.7% | 1.3% | 高频陪衬 |
| 南京路 / 陆家嘴 / 上海中心 | 35.9% | 低 | 第一梯队"巨头" |
| 城隍庙 / 田子坊 / 迪士尼 / 武康路 | 28–33% | 低 | 第二梯队 |

**本地客（C，10 条）—— 完全是另一个上海：**

| 景点 | 占答率 | 首选率 |
| --- | --- | --- |
| **共青森林公园** | **50%** | **50%** |
| M50 创意园 | 40% | 0 |
| 余德耀美术馆 | 30% | 10% |
| 外滩 | 仅 20% | 0 |

> 🔑 **关键洞察**：豆包**会区分客群**——给游客推地标（外滩/豫园），给本地人推公园/艺术区/citywalk（共青森林公园/M50）。这意味着**面向本地客和面向游客的 GEO 内容要分开做**，不能一套通吃。

**入境客（B，4 条英文）**：豆包用英文回答、推 The Bund / Oriental Pearl / Yu Garden（和中文地标一致），但 "hidden gems" 只给出 Zhujiajiao → 入境侧小众内容是更大空位（待英文引擎规模化）。

## 三、核心发现二：第一波该往哪打（内容可赢度排名）

A 段 78 个问题：**GO 39 / 候选 16 / 观望 23**。规律非常清楚——

### ❌ 别碰的（观望，23 个）：被外滩钉死的头部词
"上海必去""网红景点""一日游路线""免费景点"——**首位全是外滩**，AI 答案已固化，新内容极难挤进去。**这些词不值得做内容，硬刚是浪费。**

### ✅ 该打的（GO，39 个）：四片真机会

**🥇 机会一：亲子遛娃决策类（可赢度 96，全场最高）**
"天文馆值得去吗/怎么约""野生动物园好玩吗""科技馆 vs 自然博物馆"——豆包答得**极薄**（常只点 1 个地方），但家长**强决策意图**。→ 已写 `content/drafts/02`。

**🥈 机会二：小众 / 避坑 / 本地视角**
"小众景点""本地人才知道""不踩雷宝藏""安静小马路""不商业化"——豆包答得**分散、无标准答案**（武康路/M50/张园/召稼楼…），甚至空位。→ 已写 `content/drafts/01`。

**🥉 机会三：季节性 + citywalk + 老建筑**
"春天看花"(顾村公园)、"雨天室内"、"老洋房在哪看"(武康路)、"citywalk 路线"——答得薄、长尾多，内容能带长尾进答案。→ 已写 `content/drafts/03`。

**机会四：本地客专线（C 客群）**
共青森林公园/M50/余德耀已是本地答案头部，"周末 citywalk 小众路线"可赢度 96 → 面向上海本地人的内容是独立蓝海。→ 已写 `content/drafts/04`。

## 四、写了什么内容 + 为什么这么写（钉在证据上）

> 你要的就是这个：不光给方向，还把内容写出来，并说清"为什么写它 = 对应哪条证据"。

### 📄 `content/drafts/01_shanghai_offbeat_treasure_guide.md`（小众宝藏避坑）
- **为什么写**：小众族问题豆包答得分散、没钉死标准答案（"不商业化"甚至空位）→ 内容能定义答案。对比"上海必去"已被外滩 51% 钉死、做不动。
- **怎么做**：把豆包零散提到的 武康路/M50/张园/召稼楼/安福路/杨浦滨江 等长尾，按"老马路/艺术园区/滨水原生态"三类**结构化串成一篇**，标注是否免费/是否预约/避坑时间点。
- **诚实**：门票/闭馆/预约标"发布前核实"；客户专属信息留 `[待补]`，不编。

### 📄 `content/drafts/02_shanghai_family_kids_guide.md`（亲子遛娃全攻略）
- **为什么写**：亲子决策类可赢度 96（全场最高），豆包答得最薄（"带娃室内"只答出七宝老街、答非所问）→ 家长强意图、内容最易被采纳。
- **怎么做**：把天文馆/科技馆/自然博物馆/野生动物园按**孩子年龄 + 是否需预约**做成决策表 + 横向对比 + 行程模板，直击"选哪个/怎么约"。
- **诚实**：预约方式/票价/适龄标"发布前核实"（场馆政策常变）。

> 两篇开头都写了**「为什么写这篇 + 真实证据 ID」**，所有 11 个引用 ID 已逐个核实**确实存在**于 `evidence/captures/`（红线 #1：可回溯，不编 ID）。

## 五、诚实交代的局限（不假装）

1. **"周边古镇/两日游/自驾"显示为空位**，但这是**名单局限**——周边的乌镇/西塘/苏州不在"上海景点"白名单里，不是真内容空位。系统已自动标注"注意核实是否名单外景点"，没冒充机会。
2. **联网引用源情报暂缺**：联网接口配额墙，这轮没拿到"哪些网站内容被 AI 引用"。配额恢复后补一轮，能知道该往哪个平台铺内容。
3. **本地客(10)/入境客(4)样本小**：是试探性切片，规律方向可信，规模化需补侦察。
